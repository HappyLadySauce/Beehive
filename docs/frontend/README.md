## Beehive 前端文档总览

本目录用于描述 Beehive 新架构下的 Web 前端设计，包括：

- 面向终端用户的 **IM Web 客户端**；
- 面向运维/运营人员的 **Admin 后台控制台**；
- 前端技术栈与 UI/组件库选型；
- UX 规范与主题设计。

推荐先阅读：

- `stack-and-libraries.md`：整体技术栈与组件库评选结论；
- `client-app.md`：IM Web 客户端页面与状态设计；
- `admin-app.md`：Admin 后台页面与模块设计；
- `ux-and-theming.md`：通用交互与主题规范。
